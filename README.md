## 🧭 What this app does

jido_claw is a desktop-friendly command line app for running AI agent tasks on your computer. It helps you start an agent, give it work, and watch it act in the terminal. It uses a swarm model, so one main agent can coordinate smaller task workers.

You can use it to:

- run agent tasks from one place
- connect to more than one AI provider
- use built-in tools for files, shell work, and web tasks
- watch live status in the terminal
- manage task groups with swarm orchestration

Swarm mode lets one main agent split work across smaller agents. This is useful when a task has many parts.

Example uses:

- keep one agent focused on planning
- let other agents handle checks or file work
- track progress in one place

You do not need to know how swarm systems work. Just start the task and let the app organize the work.
